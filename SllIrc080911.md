# 会议概要 #

21点会议开始，余波担任主持人

主要讨论问题一的解决：

（1）内容整理有几种方案，手动、自动、基于社区发展过程问题

（2）需要做一些有利于今后更好整理论坛/列表资料的行为规则，具体如引导跟贴习惯、新手引导、及时WIKI等等

最关键的是形成即时整理的观念，时时整理，时时维护，所有人,都是作者和整理者，都是社区的主人，因此第二个问题也就不再需要讨论。

最后仨的建议：开始整理,大家不要想着一气儿就整理好,先各自整理自个儿有兴趣的,慢慢的,就在这过程中发现相互有同类偏好的老师,自然形成专题研究小组,自然的

GoogleGroup论坛内容整理wiki页面： http://code.google.com/p/sociallearnlab/wiki/SllGgResource

为便于大家整理互联，使用通用tag：slllist
